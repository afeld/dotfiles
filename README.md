# Aidan's dotfiles

Moving towards one-liner setup of a development machine.

## Getting Started

Fork this repo, then run the following in a terminal:

```bash
cd ~
git clone YOUR_FORK_URL.git
cd dotfiles
./init.sh
```

## Software

I use the following pieces of software in my dev setup.
